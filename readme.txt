Краткое описание каждой программы

FightWithCat - Работа с граф. интерфейсом и попытка сделать управление (но увы надо нажать ENTER для действия)

Labirint - лабиринт с рандомизацией карты. Так-же есть возможность запуска на заранее созданых картах

Chigivaras Rise - Работа с таймером, графикой и классами (Для работы нажать в любое место программы)
 
OveBookMaker - Основная задача была в разметке по номерам символов, а не по самим символам
(Каждый 255й символ - красный.Зеленый символ - "якорь"). Но также есть и другой функционал:
Кнопки наверху включают / выключают видимость разметки. Кнопка отправки отправляет весь текст
из одной части в другую, вплоть до зеленого "якоря", вставляя вотермарку. Так-же есть возможность
заменить во всех текстах ё -> е, загрузить файл в левую часть или сохранить его из правой.

BLCallHelper - Программа, которая облегчает обзвон клиентов. Работой в том месте не горжусь, но
программу мне понравилось делать и дорабатывать во время работы там.

Game 2_1 - Изучение основ Unity (В чем помогал Константин). В главном меню можно выбрать количество
игроков, а так-же количество сундуков. После начала игры каждый игрок может котырть сундук, после
чего либо отказаться (макс. 4 раза), либо согласится в суммой внутри и дать ход следующему игроку.
Побеждает игрок с наибольшим счетом.
На экране после игры кнопка [] позволяет перейти на главное меню, для изменения настроек.

SD_PROC - работа с базами данных, передачей данных между формами а так-же работой с datagrid
